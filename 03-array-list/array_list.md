## Припомняне — какво е масив?

- Масивът е **наредена колекция от елементи от един и същи тип**
- Елементите се съхраняват **последователно в паметта**
- Достъпът до елемент става чрез **индекс** (0-базиран)

```cpp
int arr[5] = {10, 20, 30, 40, 50};
//            [0]  [1]  [2]  [3]  [4]
```

---

## Масив в паметта

![array](https://www.tutorialspoint.com/data_structures_algorithms/images/arrays1.jpg)

- Всеки `int` заема 4 байта
- Адресът на елемент `i` = `базов_адрес + i × размер_на_тип`
- Затова достъпът по индекс е **O(1)**

---

## Основни операции при масив

| Операция              | Сложност | Бележка                              |
| --------------------- | -------- | ------------------------------------ |
| Достъп по индекс      | O(1)     | Директно изчисляване на адрес        |
| Търсене (линейно)     | O(n)     | Преглеждане на всеки елемент         |
| Търсене (двоично)     | O(log n) | Само при сортиран масив              |
| Вмъкване в края       | O(1)     | Ако има свободно място               |
| Вмъкване на позиция i | O(n)     | Трябва да се преместят елементи      |
| Изтриване на позиция i| O(n)     | Трябва да се преместят елементи      |

---

## Вмъкване на позиция i

```
Преди: [10, 20, 30, 40, 50]
Вмъкни 99 на позиция 2

Стъпка 1 — Преместване надясно:
[10, 20, 30, 40, 50, _]  →  [10, 20, _, 30, 40, 50]

Стъпка 2 — Запис на стойността:
[10, 20, 99, 30, 40, 50]
```

- В най-лошия случай (позиция 0) → преместваме **всички** n елемента → O(n)

---

## Предимства на масивите

- **Бърз достъп по индекс** — O(1)
- **Ефективно използване на кеша** — последователна памет
- **Проста имплементация**
- **Минимален overhead** — само данните, без допълнителни указатели

---

## Недостатъци и ограничения на масивите

- **Фиксиран размер** — трябва да знаем размера предварително
- **Скъпо вмъкване/изтриване** в средата — O(n)
- **Загуба на памет** при голям предварително заделен масив
- **Невъзможно разширяване** след създаването

```cpp
int arr[100];  // Какво ако ни трябват 101 елемента?
```

---

## Какво е ArrayList?

- **ArrayList** (динамичен масив) е структура от данни, която:
  - Използва масив вътрешно
  - Може **автоматично да се разширява** при нужда
  - Предоставя удобен интерфейс за добавяне и изтриване

- Решава основния проблем на масива — **фиксирания размер**

---

## Как надгражда масива?

Основна идея:
1. Пази масив с **капацитет** (capacity) по-голям от броя елементи
2. При добавяне — записва на следващата свободна позиция
3. При запълване — **заделя нов, по-голям масив** и копира данните

![array_list](https://assets.janbasktraining.com/blog/uploads/images/image1_1_26.jpg)

---

## Разширяване (resize)

```
Преди: capacity=4, size=4
[10, 20, 30, 40]

Добавяме 50 → няма място!

1. Заделяме нов масив с capacity=8
2. Копираме елементите
3. Добавяме 50

След: capacity=8, size=5
[10, 20, 30, 40, -, -, -]
```

- Обикновено **удвояваме** капацитета при разширяване

## Примерна имплементация на C++

```cpp
class ArrayList {
private:
    int* data;
    int size;
    int capacity;

public:
    ArrayList() {
        capacity = 2;
        size = 0;
        data = new int[capacity];
    }

    ~ArrayList() {
        delete[] data;
    }
```

---

```cpp
    void push_back(int value) {
        if (size == capacity) {
            resize();
        }
        data[size++] = value;
    }

    int get(int index) {
        return data[index];
    }

    int get_size() {
        return size;
    }
```

---

```cpp
private:
    void resize() {
        capacity *= 2;
        int* new_data = new int[capacity];

        for (int i = 0; i < size; i++) {
            new_data[i] = data[i];
        }

        delete[] data;
        data = new_data;
    }
};
```

## Пример за употреба

```cpp
int main() {
    ArrayList list;

    list.push_back(10);
    list.push_back(20);
    list.push_back(30);

    for (int i = 0; i < list.get_size(); i++) {
        cout << list.get(i) << " ";
    }
    // Изход: 10 20 30
}
```

## Сложност на операциите при ArrayList

| Операция              | Сложност        | Бележка                              |
| --------------------- | --------------- | ------------------------------------ |
| Достъп по индекс      | O(1)            | Същото като масив                    |
| Добавяне в края       | O(1) амортизирано | Рядко се налага resize             |
| Вмъкване на позиция i | O(n)            | Преместване на елементи              |
| Изтриване на позиция i| O(n)            | Преместване на елементи              |
| Търсене               | O(n)            | Линейно преглеждане                  |
| Разширяване (resize)  | O(n)            | Копиране, но рядко се случва         |

---

## Амортизирана сложност на добавяне

- **resize** се случва при: 1, 2, 4, 8, 16, ... елемента
- За n добавяния общата работа по копиране е:

```
1 + 2 + 4 + 8 + ... + n = 2n - 1
```

- Средно: **2n / n = O(1)** на добавяне
- Затова казваме: O(1) **амортизирано**

## `vector` — стандартната имплементация

- C++ предоставя `std::vector` — напълно готова и оптимизирана имплементация на ArrayList
- Включва се с `#include <vector>`

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v;

    v.push_back(10);
    v.push_back(20);
    v.push_back(30);

    cout << v[1];  // 20
}
```

---

## Основни операции на `vector`

```cpp
vector<int> v = {1, 2, 3};

v.push_back(4);       // Добавяне в края:    [1, 2, 3, 4]
v.pop_back();         // Премахване от края: [1, 2, 3]
v.insert(v.begin() + 1, 99); // Вмъкване на позиция 1: [1, 99, 2, 3]
v.erase(v.begin() + 2);      // Изтриване на позиция 2: [1, 99, 3]

cout << v[0];         // Достъп по индекс: 1
cout << v.size();     // Брой елементи:    3
cout << v.capacity(); // Текущ капацитет
```

---

## Итерация върху `vector`

```cpp
vector<int> v = {10, 20, 30, 40};

// С индекс
for (int i = 0; i < v.size(); i++) {
    cout << v[i] << " ";
}

// С range-based for (препоръчително)
for (int x : v) {
    cout << x << " ";
}
```

---

## Инициализиране на `vector`

```cpp
vector<int> a;               // Празен вектор
vector<int> b(5);            // 5 елемента, стойност 0
vector<int> c(5, 42);        // 5 елемента, стойност 42
vector<int> d = {1, 2, 3};   // С конкретни стойности
vector<int> e(d);            // Копие на d
```

## Полезни методи на `vector`

| Метод              | Описание                              |
| ------------------ | ------------------------------------- |
| `push_back(x)`     | Добавя x в края                       |
| `pop_back()`       | Премахва последния елемент            |
| `size()`           | Брой елементи                         |
| `capacity()`       | Текущ капацитет                       |
| `empty()`          | Проверка дали е празен                |
| `clear()`          | Изтрива всички елементи               |
| `front()`          | Първи елемент                         |
| `back()`           | Последен елемент                      |
| `insert(pos, x)`   | Вмъква x на позиция                   |
| `erase(pos)`       | Изтрива елемент на позиция            |

## Масив срещу ArrayList (`vector`)

| Масив                    | ArrayList / `vector`           |
| ------------------------ | ------------------------------ |
| Фиксиран размер          | Динамичен размер               |
| По-бърз (без overhead)   | Малък overhead при разширяване |
| Ръчно управление         | Автоматично управление         |
| Подходящ при известен n  | Подходящ при неизвестен n      |
